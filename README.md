# Global Wayfinding Directory

Curated global directory of elite wayfinding/signage consultancies with focus on luxury developments, flagship hospitality, premium retail, civic landmarks, and high-end mixed-use destinations.

## Scope
- Prioritize strategy-led wayfinding consultancies and environmental graphic design firms
- Focus on proven involvement in high-profile luxury or landmark projects
- Exclude generic print/sign shops unless consultancy-grade credentials are clear

## Data schema
Uses the same schema as the UK directory for compatibility with existing UI tooling.

## Trial workflow (Superpowers)
This repo is the one-repo trial sandbox for the Superpowers methodology.

### Suggested flow
1. Brainstorm target expansion region/country
2. Write explicit acceptance criteria for "god-tier" inclusion
3. Build task plan in small chunks
4. Execute via subagents with review gates
5. Commit country/region batches with evidence links

## Run locally
Open `index.html` directly or serve with any static server.
